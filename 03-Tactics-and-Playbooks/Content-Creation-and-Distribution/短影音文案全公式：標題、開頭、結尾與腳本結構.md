# 短影音文案全公式：標題、開頭、結尾與腳本結構

> **Up:** [[Home|Home]] → [[03-Tactics-and-Playbooks/Tactics and Playbooks Index|Tactics and Playbooks]] → [[03-Tactics-and-Playbooks/Content-Creation-and-Distribution/Content Creation and Distribution Map|Content Creation and Distribution]]

## 一句話總結

一套可直接照抄的文案公式庫：六種標題、四種開頭、三種結尾、每個內容主題對應的腳本結構，加上爆文寫作的四大原則與八大技巧。

## 背景

本條目將「短影音文案全公式：標題、開頭、結尾與腳本結構」整理為可重複使用的playbook，以下保留原有框架、步驟與例證。

## 執行方法

### 六種標題類型

| 類型 | 核心原理 | 範例 |
|---|---|---|
| 新知式 | 揭露新鮮、最新資訊 | 「撩妹新套路」「AI 的新用法」 |
| 提問式 | 用問題引發好奇 | 「你真的會用 AI 嗎？」 |
| 解答式 | 直接給出答案，吸引需要該答案的人 | 「冬天要買皮衣的 4 大原因」 |
| 反常式 | 打破常識，製造認知衝突 | 「為什麼明星都很窮？」 |
| 佔便宜 | 強調免費、低成本、高回報 | 「不投廣告觀看也能破百萬」 |
| 揭秘式 | 讓人想知道「背後的真相」 | 「揭秘各大品牌的供應鏈」 |

### 四種開頭方式

1. **懸念開頭** — 製造好奇心，讓觀眾非看下去不可（「花 100 塊在均消 1000 的餐廳吃飯」）。
2. **故事開頭** — 適用於重大事件、名人八卦；用情境感切入，降低說教感。
3. **極端對比** — 用強烈對比製造張力（世代對比、奢華 vs 平價）。
4. **提供價值** — 直接點出用戶痛點，提供解決方案，最適合教知識、乾貨類內容。

### 三種結尾方式

1. **金句結尾** — 3-5 秒內完成，要能單獨截取成獨立金句，概括中心思想。
2. **情緒昇華** — 觸碰人性共鳴，適合故事類、雞湯類內容。
3. **行動呼籲（CTA）：**

| CTA 目標 | 範例話術 |
|---|---|
| 按讚 | 「我每天都在用心做影片，點個愛心是我最大的動力」 |
| 留言 | 「你同意這個做法嗎？留言區告訴我」 |
| 分享 | 「把這個影片分享給你做電商的朋友」 |
| 收藏 | 「趕快把這個影片收藏起來，免得要用的時候找不到」 |

### 腳本結構全地圖

| 內容主題 | 腳本結構公式 |
|---|---|
| 曬過程（通用） | 破題 → 價值增量 → 關鍵步驟 1、2、3 → 結尾 |
| 行業揭秘 | 揭秘事件 → 事件內幕 1、2、3 → 結尾 |
| 乾貨輸出（知識萃取） | 問題 → 普通解法 → 高手解法 |
| 乾貨輸出（PREP 法） | 觀點 → 理由 → 案例 → 觀點（重申） |
| 乾貨輸出（案例分享） | 描述 → 知識點 → 應用法 → 結尾 |
| 正/反推薦 | 破題 → 價值增量 → 列點 1、2、3 → 結尾 |
| 資訊整理 | 破題 → (故事點 + 啟發點) × N → 總結 |
| 說故事（通用） | 成就/概述 → (挫折 + 轉機) × N → 體悟 |
| 熬雞湯（通用） | 雞精（重點先出）→ 說明 → 案例 → 結尾 |
| 選立場（通用） | PREP：觀點 → 理由 → 案例 → 觀點 |
| 賣產品（基礎型） | USP 賣點 → 1、2、3 → CTA |
| 賣產品（解題型） | 煩惱 → 產品 → 效果 → CTA |

**PREP 法**（通用說理框架）：P（Point 先亮出觀點）→ R（Reason 給出理由）→ E（Example 舉出案例）→ P（Point 重申觀點）。適用於教知識、選立場、熬雞湯。

**說故事腳本示例**（陳修平本人案例，濃縮版）：三年前抵押房子創業，三年營收四億後遇疫情離開自己創辦的公司（挫折）→ 前輩鼓勵下創立線上課程，談成五位老師開了第一堂課（轉機）→ 拍片燒了 150 萬且三位老師暫緩（挫折）→ 堅持做完課程，朋友介紹更多老師，一年內開出多堂課、累積 30 位夥伴超過 8 萬學生（轉機）→ 體悟：「今天很殘酷，明天更殘酷，後天很美好。」

### 爆文四大原則

1. 提供好處，拒絕業味——內容優先，銷售隱藏。
2. 結構性強，具收藏價值——讓觀眾捨不得刷掉。
3. 資訊密度高，具分享價值——讓觀眾想分享給別人。
4. 開門見山，沒有一句廢話——第一句話就要進入重點。

### 八大寫作技巧

超白話（小六檢測法）、無廢話（乾我屁事檢測法）、說人話（避免官腔行銷語言）、節奏感（句子有長有短）、畫面感（用文字讓讀者看見場景）、善用比喻、具體數字（用數字取代形容詞）、有條件地埋梗（不能分散主旨）。

### 六步寫作訓練流程

每天寫（哪怕只有一段）→ 先架構再寫文 → 下筆快速如風（初稿不求完美）→ 沉澱避免自嗨（放一放再看）→ 輸入才有輸出（持續看書看片上課）→ 建立靈感池（邊看邊收集）。

## 應用

- 寫任何一支腳本前，先確定「主題→對應腳本公式」，直接套用，而非從零構思結構。
- 用小六檢測法自我審稿：把稿子念給自己聽，換位思考一個小學六年級生能否聽懂。
- 收尾前檢查是否有明確的 CTA，並確認目標是按讚、留言、分享還是收藏——不同目標話術不同，不要籠統地說「喜歡請支持」。
