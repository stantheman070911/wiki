# 口播、拍攝與剪輯實戰技巧

## 一句話總結
新手應從口播入門——不需要華麗設備或剪輯技巧；掌握背稿/看稿技巧、基本構圖口訣與剪輯四大心法，就能開始穩定產出。

## 為什麼新手必練口播
「流量跟你付出的成本，沒有半毛錢關係。」做短影音的四個「不需要」：不需要華麗的拍攝技巧、不需要高超的剪輯技巧、不需要花很多時間拍片、不需要操盤手跟藏鏡人。「練好口播，就是普通人翻身的機會。」

## 背稿 vs 看稿
**背稿：** 事先寫好稿子 → 前一晚背起來 → 一氣呵成拍完。優點：鏡頭感最自然；缺點：準備成本高，容易忘稿。

**看稿：** 用紙本或另一台裝置（平板放在鏡頭旁）；**一秒鐘原則**——看完停一秒、講完停一秒，避免眼神飄移太明顯；**一次錄完**——有錯也繼續錄，方便後期剪輯。

## 口播表現力：常犯錯誤與提升法
常犯錯誤：語調平淡沒起伏、抑揚頓挫像演講（刻意感過重）、句子間太多卡頓、和鏡頭沒有互動、聲音後製加速（觀眾感知到不自然）、場景無聊。

| 方法 | 具體做法 |
|---|---|
| 做自己 | 先熱身，放鬆狀態 |
| 感覺隨意 | 切忌正裝坐挺，展現生活感 |
| 熟悉鏡頭 | 先用前鏡頭錄，跟鏡頭當朋友 |
| 場景自然 | 切忌大白牆，用生活感場景（酒吧、戶外、移動中） |
| 對話自然 | 請別人幫拍，降低尷尬感 |
| 口條訓練 | 從模仿開始，逐步找到自己的語感 |

## 拍攝器材清單
- **攝影：** 手機（主流、入門門檻低）／相機（進階）。
- **燈光：** 環形補光燈（基礎）／自然光利用。
- **收音（由低至高）：** 領夾麥（入門）→ 初階無線麥 → 手機槍型麥 → 專業無線麥。麥克風比攝影器材更重要，優先購入領夾麥，收音效果比手機自收音好很多且成本低。

## 構圖口訣
「開啟九宮格，額頭對上線，高清六十幀，主角放中間。」具體操作：開啟九宮格輔助線；人物頭頂對準上方橫線；拍攝格式選 1080p／60fps 以上；口播類畫面主體放正中間。

## 六種運鏡手法
自拍運鏡（一人單拍）、藏鏡人運鏡（有人幫拍隱藏拍攝者）、前後運鏡（製造空間深度感）、橫移運鏡（跟隨主體移動）、偷拍運鏡（製造真實感）、多人運鏡（多角色提升戲劇感）。

## 剪輯：Raw Style 與四大心法
整體風格：「場景自然、對話自然、後製自然」——不追求過度精緻，保留真實生活感。

| 心法 | 說明 |
|---|---|
| 黃金三秒 | 前三秒必須抓住觀眾，否則會被划走 |
| 節奏緊密 | 去掉所有停頓、贅字、空白，讓影片緊湊 |
| 小六檢測法 | 剪完讓小六生看，看不懂就重剪 |
| 字幕君出場 | 加上清晰字幕，照顧靜音觀眾 |

## 應用
- 新手第一支影片直接用手機＋純口播開拍，不必等到買齊設備——先拍出來比設備完美更重要。
- 收音優先於畫質：預算有限時，先買一支領夾麥，而非升級攝影器材。
- 剪輯完成後套用「小六檢測法」自我審核，看不懂的地方直接刪減或重剪。
