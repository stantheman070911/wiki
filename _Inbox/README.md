# Inbox

Drop unprocessed material here — transcripts, links, notes, screenshots, anything not yet distilled into a WIKI entry.

## Triage workflow

1. Review each item, identify signal vs. noise.
2. Draft an entry in the relevant domain folder using `../00-Templates/Entry-Template.md`.
3. Archive the original in `../06-Source-Library/` if worth keeping.
4. Remove the item from this inbox once processed.
