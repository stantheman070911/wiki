# 對標方法論與靈感池系統

## 一句話總結
對標＝做題前先看答案：用市場驗證過的選題與觀點降低不確定性，並用結構化的靈感池資料庫持續累積可用素材。

## 對標的本質
對標（Benchmarking）＝做題前先看答案。三大核心原則：不靠靈感、不靠猜測，只用經過市場驗證有效的方法；對標不是抄襲，抄襲無法超越——一定要加入自己觀點；做流量的核心方法論就是對標，對標有 SOP 可依循。「做內容不是創作比賽，是市場理解能力的競爭。」

這與短影音代理商方法論中的「5 倍異常值法則」（觀看數 ≥ 粉絲數 5 倍即視為值得研究的異常值）是同一套邏輯的不同表述——見 [[短影音研究引擎、鉤子與格式]]。

## 對標三步驟
```
Step 1：找選題 → 用工具找已驗證爆款（to see is to believe）
Step 2：找觀點 → 分析對標影片的觀點 + 留言區觀點 + 加入自己觀點
Step 3：改風格 → 調整語氣、替換案例，做出差異化
```
核心邏輯：用市場的數據，去提高確定性。

## 六大對標工具
| 工具 | 用途 |
|---|---|
| YouTube | 搜尋相關主題的高觀看影片 |
| Google 搜尋 | 了解關鍵字熱度與相關內容 |
| 關鍵字規劃工具 | 找熱搜關鍵字 |
| 小紅書、抖音 | 找中文語境的爆款參考 |
| 客戶 FAQ | 整理客戶最常問的問題作為選題 |
| 留言選題 | 從現有影片留言中發掘受眾需求 |

## 一抄二改三創新
1. **抄** — 直接借用已驗證爆款的結構與角度。
2. **改** — 換語氣、換案例、換切入點。
3. **創新** — 在熟悉市場後，加入獨特觀點。

## 靈感池系統
### 為什麼要建立靈感池
| 問題 | 解決方案 |
|---|---|
| 要創作時沒有想法 | 收集靈感，避免靈光一閃型創作 |
| 內容重複、沒有創新 | 區隔創新：站在巨人肩膀上做差異 |
| 創作能力沒有提升 | 練習拆解爆款，培養分析力 |
| 一個人腦力有限 | 與同學組隊，建立集體靈感池 |

「100% 的內容，別人都拍過了。很可能你的閉門頓悟，只是別人的基本功。」

### 靈感池應包含的欄位
內容主題、流量密碼、呈現形式、拍攝手法、文案撰寫（標題／開頭／結尾）、變現方式——每筆記錄都對應到這六個維度，才能持續復用。

### 建立靈感池的六種方法
與六大對標工具相同：YouTube 搜尋、Google 搜尋、關鍵字規劃工具、小紅書／抖音、客戶 FAQ、留言選題。

## 應用
- 開始任何新內容系列前，先花時間用六大工具蒐集至少 10-30 條對標素材，再動手拍攝。
- 建立一份共用的靈感池試算表（主題、流量密碼、形式、備註），持續更新而非每次重新尋找靈感。
- 套用「一抄二改三創新」時，明確區分「抄結構」與「抄價值」——只抄前者，價值必須原創。
