# 個人品牌與IP打造框架

## 一句話總結
個人品牌是你被記住的方式；用「專家型」或「成長型」人設、四步驟帳號搭建與超級符號，把自己做成可被辨識的 IP。

## 個人品牌的本質
「你是誰，比你說什麼更重要。」個人品牌＝你被人記住的方式。

| 角色 | 被記住的方式 |
|---|---|
| 坐前面的女同學 | 大學連續四年拿書卷獎 |
| 英文老師 | 教你多益考 600 分 |
| 鑽戒店 | 一生只能買給一個人 |

個人品牌四要素：你是什麼人？你擅長什麼？你的外表？你的想法？

## 人設設計原則
人設＝雙方的關係（情人、朋友、家人、老師）。最好變現的兩種人設：

**專家型** — 他懂的比你多，可以幫你刷新認知；在其領域是最頂尖的。變現能力高，但門檻也高，需要「居高思危」——持續維持專業形象。

**成長型** — 我現在不是最強，但每天比昨天好；曬自己成長痕跡、過程、學習心得。距離感比專家型更近，親和力強。原則：做低不做高，誠懇不油膩。

最佳建議：「最好的人設：做自己，放大你的美。」

## 帳戶搭建四步驟

**步驟一：帳號名稱**
命名考量要素：地區、行業、身份、暱稱、理念、記憶點。範例：「來莿桐找 Sandy」（地區＋暱稱）、「民生社區老郭」（地區＋人設）、「藍太太好生活」（人設＋理念）、「攝手黎老師」（身份＋名字）。注意事項：簡單好記、加上名字；FB/IG 較難更改名稱，TikTok 較寬鬆；已有帳號是否重開，以受眾思維判斷（現有粉絲是否符合目標受眾）。

**步驟二：敘述欄**
公式：`你是誰 + 有什麼可以炫耀 + 可以給用戶什麼價值`

優質範例：「我是賣課喬伊，陳信安、吳淡如、王金平等名人課程操盤手，教你如何從零打造一堂爆款課程。」／「我是平哥，深耕台中房產 10 年，人稱台中活地圖。給你最新買房乾貨。」

劣質反例：「精選母嬰各大商品，你想要的，這裡都有，有需求歡迎私訊。」（缺乏個人特色、過於業務性）

關鍵：說人話。

**步驟三：超級符號 IP 化**
| 元素 | 說明 |
|---|---|
| 場景 | 固定拍攝場景，強化記憶點 |
| 服化道 | 固定穿著、道具（例：白襯衫＋草帽＋雨鞋） |
| 口頭禪／Slogan | 可重複記憶的標誌性話語 |
| 主題曲／背景音樂 | 固定 BGM，形成聽覺識別 |

**步驟四：IP 規劃書**
帳號名稱、敘述欄、三個標籤（個性）、導流方式、商業模式、服化道、口頭禪／Slogan、主題曲、主要內容主題——建議在開始拍片前先填完整份規劃書。

## 應用
- 開新帳號前，先寫一份完整的 IP 規劃書，而非邊拍邊想。
- 選人設時誠實評估：你目前更接近「已經是專家」還是「正在成長」？選符合現況的人設，而非硬裝專家。
- 為自己的帳號設計至少一個超級符號（固定場景、服化道、口頭禪或 BGM 之一），提高被記住的機率。
