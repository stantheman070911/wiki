# 內容策略：七大主題與十一大流量密碼

## 一句話總結
爆款＝有用＋有趣＋有共鳴；七大內容主題提供「拍什麼」的框架，十一大流量密碼提供「怎麼讓它被看見」的鉤子元素，且內容必須與商業模式相關。

## 爆款三要素
```
爆款 = 有用 + 有趣 + 有共鳴
```
必要條件：確保內容跟自己的商業模式相關，避免流量與變現脫節。

## 七大內容主題
| 主題 | 子類型 |
|---|---|
| 曬過程 | 製作過程、改造計畫、產品測試、體驗分享、任務挑戰 |
| 說故事 | 成就故事、案例故事、他人故事 |
| 教知識 | 行業揭秘、乾貨輸出、正向推薦、反向推薦、資訊整理（懶人包） |
| 熬雞湯 | 自我成長、人際關係、處世之道 |
| POV（選立場） | 正面觀點、反面觀點、中立觀點 |
| 演劇情 | 熱門梗、情境劇 |
| 賣產品 | 展現服務、展現產品、生活觀察 |

**曬過程** — 「曬過程，就是內容變現的終極形式。」子類型：製作過程（商品／料理／藝術品流程）、改造計畫（前後對比）、產品測試（真實使用體驗建立可信度）、體驗分享（消費／旅行／服務體驗）、任務挑戰（設定目標、公開記錄）。

**說故事** — 核心公式：`成就/概述 > (挫折 + 轉機)*N > 體悟`。成就故事建立可信度；案例故事（他人案例）降低自誇感；他人故事（名人、社會事件）借勢流量。

**教知識** — 核心原則：提供乾貨，建立專家形象。行業揭秘、乾貨輸出（直接給答案／方法／清單）、正向推薦、反向推薦（不要踩的坑）、資訊整理（懶人包，幫受眾做篩選整合）。

**熬雞湯** — 不是空洞激勵，而是有具體道理支撐的觀點輸出：自我成長、人際關係、處世之道。

## 十一大流量密碼
| 密碼 | 說明 | 應用示例 |
|---|---|---|
| 成本 | 揭露真實成本，製造驚訝感 | 「在屈臣氏買完全妝花 800 元 vs. 花 1 元」 |
| 受眾 | 精準鎖定特定人群 | 「月薪 3 萬以下，必看的 3 個省錢法」 |
| 隨機 | 不可控元素，製造懸念 | 「和 100 個陌生人交換禮物」 |
| 反差 | 預期外的對比 | 「為什麼老闆都是窮鬼？」 |
| 互動 | 讓觀眾參與 | 「幫我選一件！」 |
| 熱點 | 搭載時事話題 | 「50 年老房能不能買？」 |
| 懷舊 | 觸發集體記憶 | 「789 年級生的共同回憶」 |
| POV | 強烈的個人立場 | 「我的觀點：越漂亮的女孩越沒人追」 |
| 吐槽 | 批評、踢爆 | 「揭秘裝潢公司的 3 大黑心手法」 |
| 賀爾蒙 | 顏值、美食等感官刺激 | 視覺衝擊類內容 |
| 光練不說 | 行動展示，不說廢話 | 直接上手示範，省去解說 |

（原始教材圖表中另有「貼金」——背書、認證，如「知名 XX 的御用 OOO」，實際共 12 個標籤，一併記錄於此。）

## 應用
- 為每一支影片標記「屬於哪個主題＋用了哪個流量密碼」，長期下來能看出哪個組合對你的受眾最有效。
- 產出內容前先自我檢查三要素：這支影片有用嗎？有趣嗎？看完會有共鳴嗎？三個至少要中兩個。
- 定期檢查內容主題是否偏離商業模式——曬過程、教知識類內容尤其容易「叫好不叫座」，記得把主題與變現路徑綁在一起。
