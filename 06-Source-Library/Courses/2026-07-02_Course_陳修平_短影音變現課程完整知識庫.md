## 1. 執行摘要

### 課程核心主張

短影音是當前最高性價比的**免費流量紅利**。普通人只要掌握系統性的方法論，就能用零廣告費快速累積曝光、建立個人品牌，並透過粉絲經濟實現商業變現。

### 課程三句話總結

1. **流量是商業的源頭**——不懂流量，就是沙漠上種花。
2. **做短影音不靠天分，靠方法**——對標市場、有樣學樣、區隔創新。
3. **萬事起頭難，起頭要簡單**——先拍、再優化，低流量不會失敗，但停拍會。

### 實測成果（講師案例）

| 案例 | 時間 | 成果 |
|------|------|------|
| 台灣飛人陳信安 | 2 個月 | 千萬觀看、帳號漲粉 1 萬、變現 500 萬 NTD |
| 千田愛紗 | 3 個月 | 千萬觀看、FB+5 萬 IG+3 萬、變現 7,000 萬 NTD |
| 素人學員（各行各業） | 持續耕耘 | 粉絲數從數千至數十萬不等 |

> **注意**：上述成果均為特定案例，並非保證結果。成效受個人執行力、商業模式、市場條件影響。

## 2. 核心思維框架

### 2.1 商業公式思維 vs 散點思維

陳修平強調必須以**公式思維**取代**散點思維**：

```
業績 = 流量 × 轉換率 × 客單價 × 回購率
```

每個變數都有對應優化動作：

| 變數 | 短影音對應動作 | 其他工具 |
|------|--------------|---------|
| 流量 | 短影音、SEO、KOL | 線上廣告、門口打卡點 |
| 轉換率 | 私域溝通、CTA 優化 | 增加導客誘因 |
| 客單價 | 組合套餐、增加搭配品 | 高端定位 |
| 回購率 | 官方 Line 推播、節慶通知 | 忠誠方案 |

### 2.2 流量紅利時代觀

大品牌搶佔流量從不手軟。普通人、小品牌的策略是：**抓住時代的流量紅利**。

- 第一波：FB 廣告紅利（已過）
- 第二波：直播帶貨紅利（趨緩）
- **現在**：短影音紅利（進行中）

**換算邏輯**：1,000 萬次觀看 × 0.25 NTD/次 ≈ **250 萬廣告費用節省**

### 2.3 以市場為中心 vs 以自我為中心

這是全課程最根本的思維轉換：

| 舊思維（以自我為中心） | 新思維（以市場為中心） |
|----------------------|----------------------|
| 拍我想拍的 | 拍平台算法想推的 |
| 靠靈感和直覺 | 靠數據和對標 |
| 追求創作完美 | 追求市場反應 |
| 一次做到最好 | 快速迭代，持續優化 |

> **核心格言**：「影片不是拍給觀眾看的，是拍給平台看的。」

## 3. 商業定位與變現邏輯

### 3.1 商業的三種打法

| 打法 | 優點 | 缺點 |
|------|------|------|
| **品牌經濟** | 效果深且持續、產生品牌溢價 | 耗資源、進入門檻高、效果有滯後效應 |
| **流量經濟** | 粗暴有效、快速搶佔紅利 | 廣告費高、成本綁架 |
| **粉絲經濟** | 廣告費低、黏著度高 | 易有公關危機、易模仿、護城河淺 |

**課程建議**：三種打法搭配「組合拳」使用。**短影音主打粉絲經濟**，用粉絲信任度降低廣告成本，最終以私域完成成交。

### 3.2 粉絲經濟四大好處

1. 做出差異化
2. 有持續性的免費流量
3. 提高商業價值（粉絲黏性高、可賣高單價商品）
4. 拓展銷售品類、連接資源與人才

### 3.3 四種變現方式

| 方式 | 適合對象 | 流程 |
|------|---------|------|
| **抖內（打賞）** | 直播型創作者 | 發內容 → 直播 → 抖內 |
| **業配（廣告合作）** | 粉絲達一定規模 | 發內容 → 積累粉絲 → 接業配 |
| **諮詢/服務** | 專業型、知識型 | 發內容 → 導入私域 → 諮詢成交 |
| **帶貨（商品銷售）** | 有實體或數位商品 | 發內容 → 直播/私域/官網 → 成交 |

### 3.4 哪些行業最好變現？

符合以下條件者，變現效率最高：

1. **產品即內容**——拍攝過程就是賣點展示（例：食品、美妝、手作）
2. **無地區限制**——線上課程、數位商品、電商
3. **面對終端消費者**——B2C，而非 B2B

### 3.5 沒有商業定位怎麼辦？

**最小可行商業定位三步**（講師建議）：
1. 選定一個商品
2. 狂發內容
3. **不做廣告**——用有機流量驗證市場反應

> **原則**：「想好商業定位，以終為始，趕快開始。」

## 4. 個人品牌與帳戶搭建

### 4.1 個人品牌的本質

> 「你是誰，比你說什麼更重要。」

**個人品牌 = 你被人記住的方式。**

| 角色 | 被記住的方式 |
|------|------------|
| 坐前面的女同學 | 大學連續四年拿書卷獎 |
| 英文老師 | 教你多益考 600 分 |
| 鑽戒店 | 一生只能買給一個人 |

**個人品牌四要素**：你是什麼人？你擅長什麼？你的外表？你的想法？

### 4.2 人設設計原則

**人設 = 雙方的關係**（情人、朋友、家人、老師）

最好變現的兩種人設：

**專家型**
- 他懂的比你多，可以幫你刷新認知
- 在其領域是最頂尖的
- 變現能力高，但門檻也高
- 需要「居高思危」——持續維持專業形象

**成長型**
- 我現在不是最強，但每天比昨天好
- 曬自己成長痕跡、過程、學習心得
- 距離感比專家型更近，親和力強
- 原則：做低不做高，誠懇不油膩

> **最佳建議**：「最好的人設：做自己，放大你的美。」

### 4.3 帳戶搭建四步驟

**步驟一：帳號名稱**

命名考量要素（取 C6/2 之結構）：
- 地區、行業、身份、暱稱、理念、記憶點

範例：
- `來莿桐找 Sandy`（地區 + 暱稱）
- `民生社區老郭`（地區 + 人設）
- `藍太太好生活`（人設 + 理念）
- `攝手黎老師`（身份 + 名字）

注意事項：
- 簡單好記、加上名字
- FB/IG 較難更改名稱，Tik-Tok 較寬鬆
- 已有帳號是否重開：以**受眾思維**判斷（現有粉絲是否符合目標受眾？）

**步驟二：敘述欄**

公式：`你是誰 + 有什麼可以炫耀 + 可以給用戶什麼價值`

優質範例：
> 「我是賣課喬伊，陳信安、吳淡如、王金平等名人課程操盤手，教你如何從零打造一堂爆款課程。」

> 「我是平哥，深耕台中房產 10 年，人稱台中活地圖。給你最新買房乾貨。」

劣質反例：
> 「精選母嬰各大商品，你想要的，這裡都有，有需求歡迎私訊。」（缺乏個人特色、過於業務性）

關鍵：**說人話**

**步驟三：超級符號 IP 化**

| 元素 | 說明 |
|------|------|
| **場景** | 固定拍攝場景，強化記憶點 |
| **服化道** | 固定穿著、道具（例：白襯衫+草帽+雨鞋） |
| **口頭禪/Slogan** | 可重複記憶的標誌性話語 |
| **主題曲/背景音樂** | 固定 BGM，形成聽覺識別 |

**步驟四：IP 規劃書（作業模板）**

## 5. 短影音核心方法論：對標

### 5.1 對標的本質

**對標（Benchmarking）= 做題前先看答案。**

三大核心原則：

1. **不靠靈感、不靠猜測**，只用經過市場驗證有效的方法
2. **對標不是抄襲**，抄襲無法超越——一定要加入自己觀點
3. 做流量的核心方法論就是對標，對標有 SOP 可依循

> 「做內容不是創作比賽，是市場理解能力的競爭。」

### 5.2 對標三步驟

```
Step 1：找選題
→ 用工具找已驗證爆款（to see is to believe）

Step 2：找觀點
→ 分析對標影片的觀點 + 留言區觀點 + 加入自己觀點

Step 3：改風格
→ 調整語氣、替換案例，做出差異化
```

**核心邏輯**：用市場的數據，去提高確定性。

### 5.3 六大對標工具

| 工具 | 用途 |
|------|------|
| **YouTube** | 搜尋相關主題的高觀看影片 |
| **Google 搜尋** | 了解關鍵字熱度與相關內容 |
| **關鍵字規劃工具** | 找熱搜關鍵字（Google Keyword Planner 等）|
| **小紅書、抖音** | 找中文語境的爆款參考 |
| **客戶 FAQ** | 整理客戶最常問的問題作為選題 |
| **留言選題** | 從現有影片留言中發掘受眾需求 |

### 5.4 Bonus：話題測試法

**一抄二改三創新**

1. **抄**：直接借用已驗證爆款的結構與角度
2. **改**：換語氣、換案例、換切入點
3. **創新**：在熟悉市場後，加入獨特觀點

## 6. 內容策略：七大主題與十一大流量密碼

### 6.1 爆款三要素

```
爆款 = 有用 + 有趣 + 有共鳴
```

**必要條件**：確保內容跟自己的商業模式相關（避免流量與變現脫節）。

### 6.2 七大內容主題

| 主題 | 子類型 |
|------|--------|
| **曬過程** | 製作過程、改造計畫、產品測試、體驗分享、任務挑戰 |
| **說故事** | 成就故事、案例故事、他人故事 |
| **教知識** | 行業揭秘、乾貨輸出、正向推薦、反向推薦、資訊整理（懶人包）|
| **熬雞湯** | 自我成長、人際關係、處世之道 |
| **POV（選立場）** | 正面觀點、反面觀點、中立觀點 |
| **演劇情** | 熱門梗、情境劇 |
| **賣產品** | 展現服務、展現產品、生活觀察 |

### 6.3 各主題深度說明

**曬過程**
> 「曬過程，就是內容變現的終極形式。」

子類型範例：
- **製作過程**：展示商品/料理/藝術品的製作流程
- **改造計畫**：改造前後對比（空間、形象、習慣）
- **產品測試**：真實使用體驗，建立可信度
- **體驗分享**：消費體驗、旅行、服務體驗
- **任務挑戰**：設定目標，公開記錄過程

**說故事**

核心公式：`成就/概述 > (挫折 + 轉機)*N > 體悟`

- **成就故事**：自身真實成就，建立可信度
- **案例故事**：他人案例，降低自誇感
- **他人故事**：名人、社會事件，借勢流量

**教知識**

核心原則：提供乾貨，建立專家形象。

- **行業揭秘**：揭露業內不為人知的信息
- **乾貨輸出**：直接給答案、方法、清單
- **正向推薦**：真誠推薦有效的產品/方法
- **反向推薦**：告訴受眾「不要踩的坑」
- **資訊整理（懶人包）**：幫受眾做篩選和整合

**熬雞湯**

不是空洞激勵，而是有具體道理支撐的觀點輸出：
- **自我成長**：成長過程、學習反思
- **人際關係**：職場、家庭、愛情智慧
- **處世之道**：人生哲學、價值觀

### 6.4 十一大流量密碼

| 密碼 | 說明 | 應用示例 |
|------|------|---------|
| **成本** | 揭露真實成本，製造驚訝感 | 「在屈臣氏買完全妝花 800 元 vs. 花 1 元」 |
| **受眾** | 精準鎖定特定人群 | 「月薪 3 萬以下，必看的 3 個省錢法」 |
| **隨機** | 不可控元素，製造懸念 | 「和 100 個陌生人交換禮物」 |
| **反差** | 預期外的對比 | 「為什麼老闆都是窮鬼？」 |
| **互動** | 讓觀眾參與 | 「幫我選一件！」 |
| **熱點** | 搭載時事話題 | 「50 年老房能不能買？」（根據當時政策） |
| **懷舊** | 觸發集體記憶 | 「789 年級生的共同回憶」 |
| **POV** | 強烈的個人立場 | 「我的觀點：越漂亮的女孩越沒人追」 |
| **吐槽** | 批評、踢爆 | 「揭秘裝潢公司的 3 大黑心手法」 |
| **賀爾蒙** | 顏值、美食等感官刺激 | 視覺衝擊類內容 |
| **光練不說** | 行動展示，不說廢話 | 直接上手示範，省去解說 |
| **貼金** | 背書、認證 | 「知名 XX 的御用 OOO」 |

> **注意**：原文列出 11 大流量密碼，但配合 7 大主題圖表，實際列出共 12 個標籤（含「貼金」）。此處依原始教材整理，以教材圖表內容為準。

## 7. 五種呈現形式

### 7.1 形式對比表

| 形式 | 創作難度 | 用戶喜愛度 |
|------|---------|----------|
| **情境劇** | ⭐⭐⭐⭐⭐ | ⭐⭐⭐⭐⭐ |
| **影片 + 口播** | ⭐⭐⭐ | ⭐⭐⭐⭐ |
| **圖片 + 口播** | ⭐⭐ | ⭐⭐⭐ |
| **純口播** | ⭐ | ⭐⭐ |
| **切片** | ⭐ | ⭐ |

### 7.2 各形式說明

**情境劇**：有劇情、角色、場景設定的影片。創作難度最高，但觀眾黏著度最高。

**影片 + 口播**：有真實影像素材（如活動、產品、風景），加上口頭解說。難度中等，效果好。

**圖片 + 口播**：用圖片作為視覺素材，搭配口頭說明。適合教知識、資訊整理類。

**純口播**：直接對著鏡頭說話，無其他素材。難度最低，**最適合新手入門**。

**切片**：截取長影片、演講、訪談中的精華片段。適合已有影像素材的創作者。

### 7.3 形式升級策略

> **核心原則**：形式越難，對文案的要求就越低。

- **策略一**：從純口播開始，逐步升級
- **策略二**：嘗試各種做法，找到適合自己的流量密碼
- **推論**（*推斷，非直接引述*）：新手建議從純口播或圖片+口播起步，降低執行門檻，快速積累發片習慣

## 8. 靈感池系統

### 8.1 為什麼要建立靈感池？

| 問題 | 解決方案 |
|------|---------|
| 要創作時沒有想法 | 收集靈感，避免靈光一閃型創作 |
| 內容重複、沒有創新 | 區隔創新：站在巨人肩膀上做差異 |
| 創作能力沒有提升 | 練習拆解爆款，培養分析力 |
| 一個人腦力有限 | 與同學組隊，建立集體靈感池 |

> 「100% 的內容，別ˊ人都拍過了。很可能你的閉門頓悟，只是別人的基本功。」

### 8.2 靈感池包含什麼？

靈感池是一個**結構化資料庫**，每筆記錄應包含：

1. 內容主題（7 大主題中的哪一個）
2. 流量密碼（11 大密碼中的哪幾個）
3. 呈現形式（5 種形式之一）
4. 拍攝手法（運鏡、場景）
5. 文案撰寫（標題、開頭、結尾）
6. 變現方式（關聯的商業模式）

### 8.3 靈感池範例格式

| 標題 | 連結 | 內容主題 | 流量密碼 | 呈現形式 | 備注 |
|------|------|---------|---------|---------|------|
| 有沒有被背刺的經驗？ | Link | 說故事 | 成本 | 口播 | - |
| 50 年老房能不能買？ | Link | 教知識 | 熱點 | 移動口播 | - |
| 見男友長輩，幫我選一件？ | Link | 賣產品 | 互動 | 口播 | 可替換產品 |
| 和 100 個陌生人交換禮物 | Link | 曬過程 | 隨機 | 影片+口播 | 可替換產品 |

### 8.4 六種建立靈感池的方法

| 方法 | 操作說明 |
|------|---------|
| **YouTube 搜尋** | 搜尋關鍵字，按觀看數排序，分析爆款 |
| **Google 搜尋** | 了解主題熱度、找相關文章與問題 |

| **客戶 FAQ** | 整理客戶最常問的問題（這些就是最好的選題） |
| **留言選題** | 從自己或對標帳號的留言區發掘觀眾想看的內容 |

## 9. 標題、開頭、結尾公式

### 9.1 六種標題類型

| 類型 | 核心原理 | 範例 |
|------|---------|------|
| **新知式標題** | 揭露新鮮、最新資訊 | 「撩妹新套路」「AI 的新用法」 |
| **提問式標題** | 用問題引發好奇 | 「你真的會用 AI 嗎？」「為什麼你小孩總晚睡？」 |
| **解答式標題** | 直接給出答案，吸引需要該答案的人 | 「冬天要買皮衣的 4 大原因」「抖音的 4 個底層邏輯」 |
| **反常式標題** | 打破常識，製造認知衝突 | 「為什麼明星都很窮？」「越漂亮的女孩越沒人追？」 |
| **佔便宜標題** | 強調免費、低成本、高回報 | 「不投廣告觀看也能破百萬」「不運動也能瘦身的方法」 |
| **揭秘式標題** | 讓人想知道「背後的真相」 | 「揭秘各大品牌的供應鏈」「揭秘裝潢公司的 3 大黑心手法」 |

### 9.2 四種開頭方式

**1. 懸念開頭**
製造好奇心、讓觀眾非看下去不可。
> 「花 100 塊在均消 1000 的餐廳吃飯」「1 晚 2 萬的房間是什麼體驗」

**2. 故事開頭**
適用於重大事件、名人八卦、受眾關心的人事物。
用情境感切入，降低說教感。

**3. 極端對比**
用強烈的對比製造張力。
> 789 年級對比、男女對比、戰南北、奢華 vs 平價（可製作多版本）

**4. 提供價值**
直接點出用戶痛點，提供解決方案，吸引精準垂直粉絲。
最適合教知識、乾貨類內容。

### 9.3 三種結尾方式

**1. 金句結尾**
用金句對整部影片總結，概括中心思想，激發用戶認同與互動。
- 通常在 3-5 秒內完成
- 要能單獨截取成獨立金句

**2. 情緒昇華**
觸碰人性共鳴，將情緒拉滿，激發用戶互動。
適合故事類、雞湯類內容。

**3. 行動呼籲（Call to Action）**
直接引導觀眾的下一步行動：

| CTA 目標 | 範例話術 |
|---------|---------|
| 按讚 | 「我每天都在用心做影片，點個愛心是我最大的動力」 |
| 留言 | 「你同意這個做法嗎？留言區告訴我」 |
| 分享 | 「把這個影片分享給你做電商的朋友，或許可以幫助他」 |
| 收藏 | 「趕快把這個影片收藏起來，免得要用的時候找不到」 |

## 10. 腳本結構全地圖

### 10.1 各內容主題對應腳本公式

| 內容主題 | 腳本結構公式 |
|---------|------------|
| **曬過程**（通用）| 破題 → 價值增量 → 關鍵步驟 1、2、3 → 結尾 |
| **行業揭秘** | 揭秘事件 → 事件內幕 1、2、3 → 結尾 |
| **乾貨輸出**（知識萃取）| 問題 → 普通解法 → 高手解法 |
| **乾貨輸出**（PREP 法）| 觀點 → 理由 → 案例 → 觀點（重申）|
| **乾貨輸出**（案例分享）| 描述 → 知識點 → 應用法 → 結尾 |
| **正/反推薦** | 破題 → 價值增量 → 列點 1、2、3 → 結尾 |
| **資訊整理** | 破題 → (故事點 1 + 啟發點 1) × N → 總結 |
| **說故事**（通用）| 成就/概述 → (挫折 + 轉機) × N → 體悟 |
| **熬雞湯**（通用）| 雞精（重點先出）→ 說明 → 案例 → 結尾 |
| **選立場**（通用）| PREP：觀點 → 理由 → 案例 → 觀點 |
| **賣產品**（基礎型）| USP 賣點 → 1、2、3 → CTA |
| **賣產品**（解題型）| 煩惱 → 產品 → 效果 → CTA |

### 10.2 說故事完整腳本示例

以下為教材中提供的完整故事腳本（陳修平本人案例）：

---

**（成就/概述）**
三年前我把房子抵押，借了 200 萬創業。曾經我有一家成功的電商公司，把台灣商品外銷到大馬，第三年就營收四億，拿到國內知名創投投資，在 30 歲的年紀，意氣風發。

**（挫折）**
直到遇到 Covid-19，馬來西亞經濟衰退，一年內業績砍半、和夥伴意見相左，離開了自己親手創辦的公司。在業績最好的時候，大家都叫我電商大神，離開之後，我開始懷疑自己的價值。沒了名片，我又是誰？

**（轉機）**
在最迷茫的時候，一位教育界的前輩，鼓勵我把電商的經驗，投入教育產業。於是我借了 200 萬，創立 ShiFu 線上課程，打算找各領域的大師開課。一開始靠朋友幫忙，一下子就談好五位老師，順利開了第一堂籃球課。

**（挫折）**
結果因為沒有經驗，光是拍片，就燒了 150 萬，而且還沒拍完。雪上加霜的是，原本談好的五位老師，有三位決定暫緩。當時的我，像是好不容易燃起的火苗，又被無情撲滅。有一度差點放棄，準備回去找工作。

**（轉機）**
但心裡又想，籃球課都洗下去了，就不管預算，把課程做好吧！可能是看到了我們的努力，身邊的朋友，開始幫我介紹老師。在短短的一年之內，我們就開了乃哥的麻將課、柯以柔團購課、愛紗日文課，還有王金平的溝通協調課。現在我們已經有 30 位夥伴，超過 80,000 名學生，還在持續成長。

**（體悟）**
今天很殘酷，明天更殘酷，後天很美好。如果我沒有堅持下去，可能就停在明天的晚上，等不到後天的雨過天晴。如果你遇到挫折，再堅持一下、再努力一下，最後一定會看見希望。

```
P（Point）：重申觀點，強化印象
```

適用場景：教知識、選立場、熬雞湯

## 11. 爆文寫作法

### 11.1 爆文四大原則

1. **提供好處，拒絕業味**——內容優先，銷售隱藏
2. **結構性強，具收藏價值**——讓觀眾捨不得刷掉
3. **資訊密度高，具分享價值**——讓觀眾想分享給別人
4. **開門見山，沒有一句廢話**——第一句話就要進入重點

### 11.2 八大寫作技巧

| 技巧 | 說明 |
|------|------|
| **超白話**（小六檢測法）| 讓小六生能看懂——不用術語，說人話 |
| **無廢話**（乾我屁事檢測法）| 每一句話都必須和觀眾有關，否則刪除 |
| **說人話** | 避免官腔、行銷語言、過度包裝 |
| **節奏感** | 句子有長有短，讓讀者有呼吸感 |
| **畫面感** | 用文字讓讀者「看見」場景 |
| **善用比喻** | 把抽象概念具象化 |
| **具體數字** | 用數字取代形容詞（「很多」→「3 倍」）|
| **有條件地埋梗** | 在適當時機加入趣味，但不能分散主旨 |

### 11.3 六步寫作訓練流程

1. **鍛鍊文案肌肉**：每天寫，哪怕只有一段
2. **先架構再寫文**：先列大綱，再填內容
3. **下筆快速如風**：初稿不求完美，先寫出來
4. **沉澱避免自嗨**：寫完放一放，再看才客觀
5. **輸入才有輸出**：持續看書、看片、上課，補充素材
6. **建立靈感池**：邊看邊收集，形成持續素材庫

## 12. 口播訓練：新手入門

### 12.1 為什麼新手必練口播？

> 「流量跟你付出的成本，沒有半毛錢關係。」

**做短影音的四個「不需要」**：

1. 不需要華麗的拍攝技巧
2. 不需要高超的剪輯技巧
3. 不需要花很多時間拍片
4. 不需要操盤手跟藏鏡人

> 「練好口播，就是普通人翻身的機會。」

### 12.2 背稿 vs 看稿

**背稿作法**：
1. 事先寫好稿子
2. 前一晚背起來
3. 一氣呵成拍完

優點：鏡頭感最自然
缺點：準備成本高，容易忘稿

**看稿作法**：
1. 看稿工具：紙本或另一台裝置（平板放在鏡頭旁）
2. **一秒鐘原則**：看完停一秒、講完停一秒（避免眼神飄移太明顯）
3. **一次錄完**：有錯也繼續錄，方便後期剪輯

### 12.3 口播表現力常犯錯誤

1. 語調平淡沒起伏
2. 抑揚頓挫像演講（刻意感過重）
3. 句子間太多卡頓
4. 和鏡頭沒有互動
5. 聲音後製加速（觀眾感知到不自然）

### 12.4 口播表現力提升法

| 方法 | 具體做法 |
|------|---------|
| **做自己** | 先熱身，甚至喝點酒（×2），放鬆狀態 |
| **感覺隨意** | 切忌正裝坐挺，展現生活感 |
| **熟悉鏡頭** | 先用前鏡頭錄，跟鏡頭當朋友 |
| **場景自然** | 切忌大白牆，用生活感場景 |
| **對話自然** | 請別人幫拍，降低尷尬感 |
| **口條訓練** | 從模仿開始，逐步找到自己的語感 |

**課程作業**：拍一篇「口播 + 教知識」影片，上傳學員社團。

## 13. 拍攝技巧：工具、構圖、運鏡

### 13.1 器材清單

**攝影器材**：
- 手機（主流選擇，入門門檻低）
- 相機（進階）

**燈光**：
- 環形補光燈（基礎）
- 自然光利用

**收音器材（由低至高）**：

| 等級 | 器材 |
|------|------|
| 入門 | 領夾麥 |
| 初階 | 初階無線麥 |
| 進階 | 手機槍型麥 |
| 專業 | 專業無線麥 |

> **推斷**（*非直接引述*）：初期建議購入領夾麥，收音效果比手機自收音好很多，且成本低。

### 13.2 構圖口訣

> 「開啟九宮格，額頭對上線，高清六十幀，主角放中間。」

具體操作：
- 手機設定開啟**九宮格輔助線**
- 人物頭頂對準九宮格的**上方橫線**
- 拍攝格式選**1080p / 60fps**以上
- 畫面主體放置**正中間**（口播類）

### 13.3 六種運鏡手法

| 運鏡 | 適用場景 |
|------|---------|
| **自拍運鏡** | 一人單拍、Vlog 感 |
| **藏鏡人運鏡** | 有人幫拍，隱藏拍攝者 |
| **前後運鏡** | 製造空間深度感 |
| **橫移運鏡** | 跟隨主體移動、展示空間 |
| **偷拍運鏡** | 製造真實感、記錄感 |
| **多人運鏡** | 多角色場景，提升戲劇感 |

## 14. 表現力訓練

### 14.1 第十二章表現力補充（vs 第五章口播章節）

本章節為第五章口播訓練的**進階版**，聚焦在「鏡頭前的整體表現」而非僅限口播技巧。

## 15. 剪輯重點

### 15.1 整體風格：Raw Style（自然流）

> 「場景自然、對話自然、後製自然。」

不追求過度精緻的製作感，保留真實的生活感。

### 15.2 剪輯四大心法

| 心法 | 說明 |
|------|------|
| **黃金三秒** | 前三秒必須抓住觀眾，否則他們會划走 |
| **節奏緊密** | 去掉所有停頓、贅字、空白，讓影片緊湊 |
| **小六檢測法** | 剪完讓小六生看，看不懂就重剪 |
| **字幕君出場** | 加上清晰字幕，增強資訊吸收，照顧靜音觀眾 |

> **推斷**（*非直接引述*）：黃金三秒指的是影片開頭前三秒的鉤子（hook）設計，對應「四種開頭」中的任一方式，需在最短時間內製造吸引力。

## 16. 私域經營與變現邏輯

### 16.1 公域 vs 私域

| 維度 | 公域流量 | 私域流量 |
|------|---------|---------|
| **定義** | 透過第三方平台獲得的流量 | 透過自主渠道沉澱的流量 |
| **掌控性** | 你無法掌握的流量 | 可直接與用戶互動 |
| **渠道** | Google、Meta、TikTok、YouTube | Line、EDM、官網、社團 |
| **優點** | 覆蓋廣、效果快 | 控制性高、可直接互動 |
| **缺點** | 成本逐年升高、無法精準互動 | 需長期經營、對內容運營能力要求高 |

### 16.2 做私域的三大目標

1. **留存**：讓流量有地方停泊，與客戶創造更多接觸點
2. **回購**：養成用戶使用習慣，從新粉變鐵粉
3. **裂變**：舊客拉新客，降低獲客成本

### 16.3 私域經營三段邏輯

```
引流 → 留存 → 轉換
```

| 階段 | 關鍵問題 |
|------|---------|
| **引流** | 用戶從哪來？用什麼方式來？如何提高流量？ |
| **留存** | 用戶怎麼互動？輸出什麼內容？如何活化用戶？ |
| **轉換** | 轉換的工具？轉換的節點？如何提升轉換率？ |

### 16.4 私域經營四個方向

| 方向 | 原理 | 內容 | 適合平台 | 適合行業 |
|------|------|------|---------|---------|
| **交流平台** | 讓用戶變社群，自己增加黏著度 | 新品、開箱、心得、提問 | 社團 | 服裝、母嬰、美妝、旅遊、直播 |
| **輸出乾貨** | 持續輸出有價值內容，建立專業形象 | 分享乾貨、解決用戶痛點 | Line、EDM | 知識類、母嬰 |
| **專屬顧問** | 1 對 1 服務，解決用戶急迫需求 | 新知、關懷、活動邀請 | Line | 汽車、精品、珠寶（高單價）|
| **發送優惠** | 提供專屬優惠，讓人捨不得封鎖 | 優惠資訊、折扣、上新通知 | Line、EDM | 電商、餐飲 |

> **核心公式**：「引流在公域，服務在私域，成交在全域。」

## 17. 持續創作之路：心態與常見陷阱

### 17.1 陳修平親踩的六大坑

1. **思想的巨人，行動的侏儒**——想太多，做太少
2. **追求規格，反而拖垮效率**——設備、場景、剪輯過度追求完美
3. **偶包作祟，在意外表、環境等不重要的事情**——怕醜、怕被笑
4. **自尊作祟，追求高大上專家形象**——過度包裝，失去真實感
5. **專業作祟，講一些沒人聽的東西**——用術語、講太深，觀眾聽不懂
6. **自滿作祟，停止追求更有效方法**——過度自信，停止學習

### 17.2 最重要的一句話

> **「低流量不會失敗，但停止拍片會。」**

### 17.3 正確使用這套課程的方式（完整五步）

1. **看完課程** → 吸收核心概念，建立認知
2. **完成作業** → IP 規劃書、說故事/教知識腳本
3. **團體輔導課** → 專業操盤手批改，吸收反饋
4. **拍攝剪輯課** → 拍攝剪輯實作，踏出第一步
5. **使用線上諮詢** → 問題總在執行後，一路有人陪

## 附錄：常見問題 FAQ（整理自教材）

### Q：不露臉可以拍嗎？
**A**：課程未明確說明。從教材推斷（*推斷非直述*）：切片、圖片+口播可以不露臉；口播類難以完全不露臉。建議先從露臉的口播開始，突破心理障礙。

### Q：第一支影片要拍什麼？
**A**：拍「口播 + 教知識」——選一個你熟悉的問題，直接回答客戶最常問的一個 FAQ。

### Q：一個人要怎麼拍攝？
**A**：純口播可以自拍（前鏡頭）。需要有人幫拍的場景可以用手機支架或請朋友幫忙。

### Q：面對鏡頭很尷尬怎麼辦？
**A**：先熱身、先用前鏡頭讓自己習慣鏡頭。請別人幫拍可以降低尷尬感。從模仿開始，逐步找到自己的表現方式。

### Q：要怎麼選拍攝設備？
**A**：手機即可入門。麥克風比攝影器材更重要，優先購入領夾麥。

### Q：哪些類型的題目會紅？
**A**：用對標工具找已驗證爆款，加上 11 大流量密碼中的至少一個，確認內容符合「有用、有趣、有共鳴」三要素。
