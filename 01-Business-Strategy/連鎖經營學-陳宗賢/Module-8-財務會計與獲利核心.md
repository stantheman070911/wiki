# 連鎖經營學 · Module 8：財務會計與獲利核心

> 系列導覽：[[README|總覽與方法論]]　｜　上一模組：[[Module-7-招商加盟與組織人才|Module 7]]

## 一句話總結
連鎖的獲利核心來自規模經濟、品牌優勢、營運優化三者的疊加,但前提是總部與門店櫃各自都有清楚的會計制度、預算編制與日結損益——多算勝,少算不勝,而況於無算乎。

## 情境
本模組收尾整個課程,把前七模組的商業邏輯落實成具體的財務制度：總部該用什麼公司型態、怎麼編預算;門店櫃（不論直營、加盟、內創）該怎麼規劃開辦費與日常損益;最後回到連鎖業獲利的三大基礎做總結。

## 關鍵洞察
- **總部公司成立必為股份有限公司。** 三種公司型態比較：商號（不具法人資格,負無限責任,不可公開募集資金）;有限公司（具法人資格,股東依出資額負有限責任,股權轉讓須經其他股東同意,不可公開發行股票）;股份有限公司（具法人資格,股東依持股比例負有限責任,可發行股票募集資金,股份轉讓相對自由）。
- **企業九大循環：** 採購、銷貨、生產、固定資產、研發、人力資源、融資、資訊、商開——總部會計制度須能對應這九大循環做理帳、結帳、轉帳與應收應付管理。
- **零基預算制的核心精神：沒有做事就不要花錢;打算花多少錢,做出什麼結果？** 執行重點：不能用過去數據推估,必須依執行計畫編專案預算。（對應範本：`陳宗賢的連鎖經營學｜8-1 年度公司費用預算彙總表.xlsx`、`｜8-1 年度公司損益預算表.xlsx`）
- **新創公司存活率的現實：** 超過 1 年 7%、超過 3 年 5%、超過 5 年僅 1.7%——這組數字是門店櫃開辦預算規劃時該有的風險意識背景。
- **多算勝,少算不勝,而況於無算乎——出自《孫子兵法·始計篇》。** 這句話被用來強調開店前的損益試算與資金運用表規劃,而非邊做邊算。
- **不論規模都必須謹慎管理零用金與損益;不會管理,再好的策略都難以發揮,甚至適得其反。**
- **總部與加盟店的兩大金流型態不同：** 總部控管營收（加盟店營收全數繳回總部,總部統一結算貨款並撥款給加盟店應得款項）vs 加盟店自主管理營收（加盟店自行保留營收,每月依應付金額支付貨款給總部）。
- **連鎖業獲利的三大基礎：規模經濟（店點越多,營收成長動能越強）、品牌優勢（品牌是被信賴的關鍵）、營運優化（管理才是獲利來源）。** 三者缺一不可——只有店多沒有品牌信任,或只有品牌沒有營運效率,都無法把獲利落實。

## 框架／模型
**總部財務管理重點的完整鏈路：** 年度經營計畫與預算編制 → 整建財會權責表（依階層劃分提案人與簽核金額,如基層主管管銷售金額、中階主管管用人留才、CEO 與老闆管重大決策）→ 預估與結總（整建總分公司經營統計表、整建總部資金運用表）→ 每日稽核（全連鎖收支彙總）。

**資金運用表的邏輯結構：** 上期結轉 ＋ 當期現金收入（應收帳款現金、應收票據、其他）－ 當期支出（應付帳款、應付費用、應付票據）＝ 餘額 → 對策（還款／轉存／借款／融資）→ 本期結轉。判斷準則：本期結轉金額須大於未來三個月的支出合計,否則需要提前規劃融資或催收。

**門店櫃財務會計規劃的四個層次：**
1. **開辦費預算規劃** — 依坪數分級（A 級店 100 坪以上、B 級店 50–60 坪、C 級店 20–30 坪）規劃對應開辦費;開辦費項目含加盟金（授權品牌）、保證金（貨款擔保、違約扣抵）、裝修費（店面裝潢）、預備三個月周轉金（開店初期開銷）。
2. **經營目標與損益規劃** — 對照資產負債表、損益表、現金流量表三張基本報表。
3. **營收與費用金流分立** — 避免營收與費用混在同一筆現金流中對帳,降低舞弊與誤植風險。
4. **零用金定義規劃** — 例：定額零用金 3,000 元,門店櫃當週花費 1,750 元後向總部申報,總部審核並撥補 1,750 元回補至滿額 3,000 元。

**加盟店 vs 內創店的財務規範差異：**
| 項目 | 加盟店 | 內創店 |
|---|---|---|
| 登記型態 | 有限公司型態 | 有限公司型態 |
| 股權結構 | 加盟者全資 | 總部 60% ＋ 內創店負責人 40%（登記負責人為內創店負責人,總部以委任經理人契約聘雇） |
| 創業金規定 | 加盟金＋保證金＋裝潢金＋周轉金（授權費/輔導費涵蓋在加盟金中） | 同左,籌措管道另有總部提供相對創業基金、與租賃公司合作、青創貸款、標會 |
| 日/週結 | 應於隔日匯轉 | 應於隔日匯轉 |
| 月結貨款 | 於次月 5 日前匯轉 | 於次月 5 日前匯轉 |
| 每日/每月結損益 | 是 | 是 |

**與租賃公司合作的槓桿優勢：** 設備由租賃公司代為出資購置,無須一次性投入大額資金,降低初期資金壓力;依 3–5 年合約分期攤還,全數清償後辦理產權移轉——這是內創店/加盟店取得設備而不稀釋現金流的常見槓桿工具。

**門店櫃經營損益日報的標準科目：** 銷貨淨額 － 銷貨成本／轉撥價 ＝ 銷貨毛利 － 管銷費用合計 ＝ 營業收益 ＋ 營業外收入 － 營業外支出 ＝ 本期損益。每日與每月同時累計,掌握門店櫃本日、本月是否賺錢。（對應範本：`陳宗賢的連鎖經營學｜8-4 日報表.xlsx`、`｜8-4 門店櫃經營損益日報表.xlsx`、`｜8-1 經營日報表.xlsx`）

**獲利創造的核心模式與關鍵策略：**
- 核心模式三支柱：價值鏈優化（Value Chain）、展店與加盟模式、多元化收入來源。
- 獲利管理關鍵策略：短鏈即時自動撥補（見 Module 6）、依商圈差異配置不同商品（見 Module 5）、員工激勵與內部創業（見 Module 2、7）、降低人員異動率。
- 員工激勵制度公式：獲利 × 20%–33% ＝ 盈餘分配——先掌握獲利絕對值,再決定分配百分比（呼應 Module 2「當一個算錢的老闆,不要當賺錢的老闆」）。

## 應用
- 開店前用零基預算制編列開辦費與首年費用預算,拒絕「用去年數字往前推估」的懶惰做法。
- 每日／每月同步產出門店櫃經營損益日報,把「本日是否賺錢」變成每天都能回答的問題,而非月底才驚訝。
- 依加盟店 vs 內創店的股權與金流差異,分別設計對應的會計科目表與結算流程,不要用同一套規則套用在兩種不同性質的門店上。
- 設備投資優先評估租賃合作的可行性,把一次性資本支出轉換成分期攤還,保留現金用於展店。
