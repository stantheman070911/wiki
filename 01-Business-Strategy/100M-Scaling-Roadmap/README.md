# $100M 擴張路線圖 · 總覽與方法論

## 一句話總結
每一個成長階段都由「單一綁定約束」定義——你的工作不是同時做好每件事，而是辨識自己所處階段的那個唯一瓶頸、只解它，然後畢業到下一關；每過一關，你的角色本身都必須改變。

## 這是什麼
Alex Hormozi / Acquisition.com 的 230 頁擴張指南，把一家公司從「一個人加一個點子」到「250–500 人、需要靠大賭注才能再成長」的歷程，拆成 0–9 共十個關卡。本系列為每一關各建一份深入條目；本頁是總覽、方法論與索引。

## 核心方法論：單一約束 + 角色轉換 + 畢業標準
每一關都具備三個要件：

1. **一個綁定約束（bottom line）** — 此刻唯一真正卡住成長的問題。其他問題都是干擾，或是「下一關才該處理的問題」。
2. **一次角色轉換（your role）** — 從研究者 → 創業者 → 執行者 → 訓練者 → 經理 → 總監 → 領導者 → 高階主管 → 總裁 → 董事長。多數卡關，是因為創辦人不肯放下上一關的角色。
3. **一條畢業標準（graduate by）** — 一個明確、可判斷的訊號；達成即進入下一關。

每一關都用同一組功能面檢視，並隨規模擴大而增加：早期是**產品、行銷、銷售、客服、技術、金錢、法務／自保**；中後期再加入**團隊／招募、人資、系統與資安**。原則：只配置精力去解「當前關卡」的約束，不要提前借用下一關的問題來焦慮，也不要停留在上一關的舒適角色。

## 貫穿全程的重複主題
- **不可規模化地把事做對（do things that don't scale）** 在早期關卡反覆出現：親自把每位客戶服務到滿意，即使沒效率。
- **約束會移動：** 早期瓶頸在「有沒有需求／收不收得到錢」，中期移到「一個人做不完／每客戶賺太少」，後期移到「組織失序／沒有專才／找不到下一條成長曲線」。
- **角色不轉換是頭號死因：** 5–9 人了還在當執行者、50–99 人了還想樣樣自己懂，都會卡死。
- **現金與 LTV 是燃料：** Stage 5 之後，延長 LTV（每客戶賺更多）換來的現金，才是加碼廣告與招募高階人才的本錢。

## 注意事項
本路線圖是 Acquisition.com 的自有框架，內文夾帶導向其自家產品（Skool、社群課程）的行銷置入，已在各條目中略去。階段人數與達標比例為美國概略統計，屬啟發性數字。框架真正的價值在「單一約束＋角色轉換＋畢業標準」這組心智模型，而非把人數當硬性分界。
